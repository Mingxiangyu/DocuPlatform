## 🛠️ 技术架构

- **框架**: Vue 3 + TypeScript + Composition API
- **构建工具**: Vite + Vitest
- **样式**: Tailwind CSS + 设计令牌系统
- **状态管理**: Pinia

- **数据库**: PostgreSQL + Prisma ORM
